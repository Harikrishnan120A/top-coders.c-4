## 🚀 **QUICK START**
```bash
# 1. Extract the zip file
unzip identity-service-v6-complete.zip
cd identity-service-v6

# 2. Install dependencies
npm install

# 3. Set up environment

# 4. Set up database
npm run prisma:generate
npm run prisma:migrate

# 5. Start the service
npm run dev
```
